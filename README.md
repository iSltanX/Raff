# رفّ (Raff)

مدير حافظة شخصي، محلي بالكامل، يعمل بالكيبورد — لنظام macOS.

A personal, **local-only**, keyboard-first clipboard manager for macOS.
Two layers: an automatic **Recent** history of everything you copy, and a
**Pinned** shelf for the prompts, snippets, and links you reuse daily.
No account, no cloud, **zero network calls** — every byte stays on your Mac.

Built with **Tauri v2**: Rust backend + vanilla HTML/CSS/JS frontend (no
framework), styled after the رفّ visual identity (warm paper & wood, honey
accent, Cairo + Almarai, full RTL, light & dark).

## Build & run

Requirements: macOS 12+, Rust (stable), Node 18+, Xcode Command Line Tools.

```sh
npm install
npm run tauri dev     # run in development
npm run tauri build   # produce Raff.app + .dmg (src-tauri/target/release/bundle/)
npm test              # frontend logic tests (node --test)
cargo test --manifest-path src-tauri/Cargo.toml   # backend tests
```

رفّ is a background **menu-bar app** — it has no Dock icon. Look for the
shelf icon in the menu bar.

## Usage

| Action | Key |
| --- | --- |
| فتح/إغلاق اللوحة | **⇧⌘V** (configurable in الإعدادات) |
| تنقّل | ↑ ↓ (PageUp/PageDown للقفز) |
| لصق في التطبيق السابق | **⏎** |
| لصق كنص عادي | **⌥⏎** |
| تثبيت / إلغاء تثبيت | **⌥P** |
| بحث | اكتب مباشرة، أو **⌘F** |
| نسخ دون لصق | **⌘C** |
| حذف العنصر المحدد | **⌘⌫** |
| إغلاق (أو مسح البحث أولًا) | **esc** |

Typing filters instantly. The panel opens centered on the screen your cursor
is on and focuses the search field immediately.

## Permissions

- **تسهيل الوصول (Accessibility)** — required only to synthesize ⌘V so رفّ can
  paste into the previous app. Without it, pressing ⏎ still copies the item to
  the clipboard and you paste manually. رفّ asks calmly on first run and opens
  the right System Settings pane (الخصوصية والأمان ← تسهيل الوصول).
- Nothing else. No network entitlements, no analytics, no accounts.

## Privacy model

- 100% on-device. History, pinned items, and settings are plain JSON in
  `~/Library/Application Support/com.raff.app/` (images as PNG files alongside).
- Content flagged `org.nspasteboard.ConcealedType` or
  `org.nspasteboard.AutoGeneratedType` (password managers) is **never stored**.
- Any app can be excluded by bundle id (الإعدادات ← الخصوصية).
- "Learning" in v1 is silent signal logging only (copy/paste counts, last use)
  — inspect it via الإعدادات ← التعلّم ← «عرض ما تعلّمه رفّ», disable it, or
  wipe it at any time.

## Project layout

```
  index.html         floating panel        js/panel.js

  src/monitor.rs     pasteboard polling & capture

  src/panel.rs       non-activating NSPanel + vibrancy
  src/tray.rs        menu-bar icon & menu
  src/macos.rs       AppKit boundary (pasteboard, AX)
  src/commands.rs    least-privilege IPC surface
tests/               frontend logic tests
R/                   visual identity sources (fonts, icon SVGs, reference)
```
